## ğŸ“˜ Concepts Covered (Important Points)

1. **Abstract Base Class (`ABC`)**
   - Used to define a blueprint for other classes  
   - Forces subclasses to implement certain methods  

2. **Inheritance**
   - Child classes (`UpCounter`, `DownCounter`) reuse code from `Counter`  

3. **Method Overriding**
   - Each subclass overrides the `count()` method  

4. **Encapsulation**
   - Data like `start`, `value`, and `step` are kept inside objects  

5. **Polymorphism**
   - The main program handles different counter types uniformly
